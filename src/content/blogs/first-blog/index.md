# 我们的第一篇社团技术博客

你好，开源社区！👋 

为了更好地管理我们的学生社团内容并且极速部署，我们决定采用 **Astro** 结合 **TailwindCSS** 进行重构。

## 为什么选择 Astro？

Astro 最吸引我们的点在于它的 `Content Collections` (内容集合) 概念，以及极快的响应速度（0 Javascript 运行在前端）。这样不管是新闻推文还是博客，都可以非常方便地使用 Markdown 来撰写。

更棒的是它完美支持图片渲染和 Tailwindcss：

![Coding Image](/images/cover.jpg)

我们期待在接下来的日子里，用这套系统发布更多社团里的：
- 优秀的个人项目展示
- 比赛经验分享
- 代码教程与小技巧
